To-Do App

Aplicativo simples de lista de tarefas, com funcionalidades básicas de autenticação, como login, auto-cadastro, ativação de usuário com código enviado por e-mail, recuperação de senha com código enviado por e-mail atualização de dados do usuário (profile) e atualização de senha com usuário logado.

Além das funcionalidades citadas o app fornece a possibilidade de criação e manutenção de tarefas e listas de tarefas, com criação, edição, deleção de tarefas, deleção de listas com suas tarefas, completar tarefas individualmente ou por lista e marcar como incompletas, além de mover tarefas de uma lista a outra.
