To-Do App - Lista de Tarefas

Technologias utilizadas

O projeto foi desenvolvido de forma a ter um front-end e back-end segregados, onde o front-end consome a API disponibilizada pela aplicação back-end que armazena as informações em banco de dados MySQL. Os projetos então foram separados e criados com linguagens diferentes ou Frameworks Diferentes, como Angular para o front-end e Java (SpringBoot) ou GO para o back-end.
