To-Do App

Simple to-do list application, with basic authentication features, such as login, self-registration, user activation with code sent by email, password recovery with code sent by email, user data update (profile) and password update with logged in user.

In addition to the features mentioned, the app provides the possibility of creating and maintaining tasks and task lists, with creation, editing, deletion of tasks, deletion of lists with their tasks, completing tasks individually or by list and marking them as incomplete, in addition to moving tasks from one list to another.
